- **Category**: Developer Tools / Productivity

## Detailed Description (Long)
**XHS Tool Pro** 是一款专为小红书创作者和开发人员设计的效率工具。它能够帮助您在复杂的创作流程中快速获取必要的数据支持。

### 主要功能：
1. **流量限流实时监控**：在创作者中心后台自动标注笔记的流量层级（正常、限流、屏弊等），基于 API 数据，提供比前端显示更深层的数据透视。
2. **身份令牌管理**：一键提取并解析当前登录的 Cookie 信息（User ID, Session, Device A1），支持单项字段快捷复制，极大简化了自动化脚本的调试过程。
3. **专业级 UI 体验**：采用极简 SaaS 风格设计，无干扰、无广告、纯净高效。

### 安全声明：
本插件仅在本地运行，不对外传输您的任何 Cookie 或敏感数据。所有分析逻辑均在您的浏览器本地完成。
