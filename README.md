**XHS Tool Pro** 是一款开源的小红书创作者辅助工具，旨在通过数据透视与自动化支持，帮助创作者更高效地管理自己的账号资产。

## ✨ 核心特性

- **流量限流监控**：实时拦截小红书后台 API 数据，自动标注笔记的流量分级（正常、限流、屏弊）。采用了 v1.3.0 物理索引同步技术，确保在所有视图下 100% 准确对齐。
- **身份令牌管理**：一键解析当前登录的 Cookie 信息。支持提取 User ID、Session 等关键字段，并提供独立快捷复制按钮，助力自动化脚本调试。
- **专业级 UI/UX**：基于 SaaS 设计标准构建的极简专业界面，支持单项字段快速复制与原始数据预览方案。
- **隐私至上**：代码完全开源，且所有操作均在本地浏览器端完成，不采集、不上传任何用户信息。

## 🛠 安装指引

1. 下载本仓库的最新 [Release](https://github.com/aaronwu/xhs-tool-pro/releases) 包并解压。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/` 页面。
3. 开启右上角的 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序”**，选择解压后的文件夹。
