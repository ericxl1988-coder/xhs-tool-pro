# 小红书工具箱 Pro - 版本迭代记录 (CHANGELOG)

本文档记录了小红书工具箱从初版至今的所有核心变更与技术突破。

---

## 🚀 最新版本：v1.3.0 (2026-03-11)
**定位：全链路诊断与数据池同步版**
- **[核心修复] 数据池合并算法**：通过 `Map` 集合实现增量抓取。解决了因接口分片加载（如每次只下发 5 条）导致旧标签数据被覆盖的问题。
- **[技术突破] 物理索引对齐**：放弃不稳定的文本匹配，采用 API 顺序与 DOM 顺序 1:1 动态挂载，彻底解决特殊字符笔记不显标的问题。
- **[全链监控]**：在控制台引入 `[XHS-Data]` 和 `[XHS-Logic]` 日志体系，实时显示抓取数 vs 页面展示数，实现问题透明化。

## 🛠 历史版本回顾

### v1.2.x 系列：UI 体验与精准匹配攻坚 (2026-03-11)
*   **v1.2.6 - Emoji 免疫系统**：引入正则过滤器，在匹配时自动剔除标题中的表情符号和特殊字符。
*   **v1.2.5 - 模糊前缀算法**：针对长标题被截断（带省略号）的情况，改为前 10 字符匹配逻辑。
*   **v1.2.4 - UI “手术级”避让**：将标签挂载点由“标题区”强行移至“封面图左上角”，彻底释放所有标题阅读空间。
*   **v1.2.1 - 视觉精致化**：引入半透明胶囊样式与磨砂背景（Backdrop-filter），提升在高光封面下的识别率。
*   **v1.2.0 - 地毯式扫描引擎**：针对手机模拟器下的扁平化 DOM 结构，引入 ID + 标题的双重搜索机制。

### v1.1.x 系列：稳定性与 Manifest V3 深度适配 (2026-03-10)
*   **v1.1.9 - 环境灾难防御**：通过实时监控 `chrome.runtime.id`，根本性解决了插件更新后常见的 `Extension context invalidated` 红色报错。
*   **v1.1.7 - 分级映射修正**：对限流等级进行精准硬编码（4:正常, -1:轻度, -102:严重），确保数据真实可靠。
*   **v1.1.4 - 双世界隔离 (Dual-World)**：将拦截逻辑移至 `MAIN` 环境（Hook API），将 UI 逻辑留在 `ISOLATED` 环境（安全注入），符合谷歌 V3 安全标准。
*   **v1.1.3 - 僵尸脚本清理**：增加了对旧版残留 Script 标签的自动清理逻辑，防止更新后加载多个拦截器导致冲突。

### v1.0.x 系列：核心能力构建 (2026-03-10)
*   **v1.0.5 - 限流检测原型**：初步实现 API 拦截，能抓取到笔记的 `level` 字段并进行控制台打印。
*   **v1.0.0 - 诞生版本**：
    *   核心功能：小红书全量 Cookie 一键提取。
    *   UI 设计：采用小红书品牌色系，实现 Popup 弹窗与一键复制功能。
